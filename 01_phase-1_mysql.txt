# Lecture 1

## Sabse pehle SQL hota kya hai?

SQL ka full form

> **Structured Query Language**

Ab isko word by word samajh.

### Structured

Matlab

Data ko ek proper format me rakhna.

Example

Student

| ID | Name    | Age |
| -- | ------- | --- |
| 1  | Sandeep | 20  |
| 2  | Rahul   | 21  |

Ye structured data hai.

---

### Query

Query matlab

👉 Database ko diya gaya instruction.

Jaise

> "Mujhe saare students dikhao."

Ye ek query hai.

Ya

> "Ek naya student add karo."

Ye bhi query hai.

---

### Language

Computer ko samjhane wali language.

Isliye

SQL = Database se baat karne ki language.

# Homework (Lecture 1)

Aaj sirf ye commands Workbench me likho aur execute karo:

```sql
CREATE DATABASE school;
```

Agar error aaye ki database already exist karta hai, to ye try karo:

```sql
CREATE DATABASE college;
```

Phir left side ke **SCHEMAS** panel me right-click karke **Refresh All** karo. Tumhe naya database dikh jayega.
